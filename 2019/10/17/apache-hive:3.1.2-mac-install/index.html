安装Hive

brew安装

此命令会把hive依赖的hadoop安装，所以就不需要单独进行安装hadoop
该命令默认安装的版本较新，我的hive是3.1.2，hadoop是3.2.1，安装位置：/usr/local/Cellar/hive/
